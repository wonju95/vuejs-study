뷰 컴포넌트 : 컴포넌트는 화면의 영역을 구분하여 개발할 수 있는 뷰의 기능(재사용성이 올라간다.)
컴포넌트 통신 방식 : 뷰 컴포넌트는 각각 고유한 데이터 유효 범위를 갖는다.
    상위에서 하위로는 데이터를 내려줌, 프롭스(props) 속성
    하위에서 상위로는 이벤트를 올려줌, 이벤트 발생(Event  Emit)
